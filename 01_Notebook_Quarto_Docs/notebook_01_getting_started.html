💡 Si vienes de R, esto te sonará familiar

Python y R son ambos lenguajes interpretados de alto nivel. La diferencia principal es que **Python no necesita `<-` para asignar** (usa `=`), **no necesita `library()` para lo básico**, y la **indentación no es opcional** — es parte de la gramática del lenguaje.

1.1 Tu Primer Programa

En R probablemente escribiste print("Hola") o simplemente "Hola". En Python, print() es siempre explícito.

# La función print() muestra texto en pantalla
print("¡Hola, Python!")
print("Bienvenido al mundo de la programación con Python")

R vs Python — Imprimir en pantalla

Tarea En R En Python
Imprimir texto print("Hola") o solo "Hola" en consola print("Hola")
Imprimir número print(42) print(42)
Imprimir múltiples valores cat("a", "b", "\n") print("a", "b")
Concatenar texto al imprimir paste("Hola", nombre) f"Hola {nombre}"

⚠️ Error común: olvidar los paréntesis

En R puedes escribir `print "Hola"` en versiones antiguas, pero en Python **los paréntesis son obligatorios** siempre. 
# ❌ Esto genera un SyntaxError en Python
print "Hola"

# ✅ Correcto
print("Hola")

1.2 Python como Calculadora

Las operaciones aritméticas son casi idénticas a R, con una diferencia importante: el operador de potencia.

# Operaciones aritméticas básicas
print(2 + 3)      # Suma
print(10 * 5)     # Multiplicación
print(2 ** 10)    # Potencia: 2^10 — en Python es ** no ^
print(17 / 3)     # División (siempre retorna float)
print(17 // 3)    # División entera (floor division)
print(17 % 3)     # Módulo: el RESTO de la división

R vs Python — Operadores aritméticos

Operación En R En Python
Suma 5 + 3 5 + 3
Resta 5 - 3 5 - 3
Multiplicación 5 * 3 5 * 3
División 5 / 3 5 / 3
Potencia 5 ^ 3 o 5 ** 3 5 ** 3 ← solo este
División entera 5 %/% 3 5 // 3
Módulo 5 %% 3 5 % 3

⚠️ Error común: usar ^ para potencias

En R usabas `^` para potencias. En Python `^` existe pero hace algo completamente distinto (XOR a nivel de bits). **Siempre usa `**` para potencias en Python.** 
# ❌ En Python ^ NO es potencia
print(2 ^ 10)   # Imprime 8, no 1024 — resultado sorprendente

# ✅ Correcto
print(2 ** 10)  # Imprime 1024

🔍 ¿Qué hace la división entera `//`?

La división entera (`//`) descarta la parte decimal y siempre redondea **hacia abajo** (floor). Es equivalente al `%/%` de R. 
print(17 // 3)   # 5  → porque 17 / 3 = 5.66... → se queda con 5
print(-17 // 3)  # -6 → porque -5.66... redondea hacia abajo a -6

💡 Buena práctica

Comenta el **por qué**, no el **qué**. El código explica qué hace; el comentario explica por qué lo hace. 
# ❌ Comentario inútil (solo repite el código)

# ✅ Comentario útil

Ejercicio 1.2 — Calculadora Geométrica

Calcula el área y perímetro de un círculo con radio 7. Usa 3.14159 para π.

# Completa el código
radio = 7
pi = 3.14159

# Escribe aquí tus fórmulas:
area = ___
perimetro = ___

print(f"Radio: {radio}")
print(f"Área: {area:.2f}")
print(f"Perímetro: {perimetro:.2f}")

✅ Solución

radio = 7
pi = 3.14159
area = pi * radio ** 2       # pi * r^2
perimetro = 2 * pi * radio   # 2 * pi * r
print(f"Área: {area:.2f}")
print(f"Perímetro: {perimetro:.2f}")

Ejercicio 1.3 — Conversión de Temperatura

En R usarías (fahrenheit - 32) * 5/9. Traduce esa fórmula a Python.

# Convierte 98.6°F (temperatura corporal) a Celsius
fahrenheit = 98.6

# Escribe la fórmula: (F - 32) * 5/9
celsius = ___

print(f"{fahrenheit}°F son {celsius:.1f}°C")

✅ Solución

fahrenheit = 98.6
celsius = (fahrenheit - 32) * 5 / 9
print(f"{fahrenheit}°F son {celsius:.1f}°C")
# Output: 98.6°F son 37.0°C

Ejercicio 1.4 — Detecta el Error

El siguiente bloque tiene 3 errores. Encuéntralos y corrígelos.

# ⚠️ Este código tiene errores — encuéntrarlos
Print("Bienvenido a Python")
resultado = 2 ^ 8
print("2 elevado a 8 es: " + resultado)

✅ Solución y explicación

**Error 1:** `Print` con P mayúscula → Python distingue mayúsculas/minúsculas. La función es `print`. **Error 2:** `2 ^ 8` → En Python `^` es XOR, no potencia. Debe ser `2 ** 8`. **Error 3:** `"texto" + resultado` → No puedes concatenar un string con un número directamente. Usa f-string: `f"2 elevado a 8 es: {resultado}"`. 
# ✅ Versión corregida
print("Bienvenido a Python")
resultado = 2 ** 8
print(f"2 elevado a 8 es: {resultado}")

💡 Truco: `*` con strings

En Python puedes multiplicar un string por un número: `"=" * 42` repite el carácter 42 veces. En R el equivalente sería `strrep("=", 42)`.
